# india-agri-price-pipeline
# India Agriculture Market Price Pipeline

Real-time analysis of daily crop prices across 819 markets 
and 26 states in India using government APMC data.

## Key Finding — Day 1
Arecanut price gap of ₹41,615 per quintal between 
best and worst market on same day (21 April 2026).

Rashi variety fetches 3.7x more than Sippegotu 
in the same market on the same day.
